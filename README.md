### How to use:
* setup modules from package manager: `yarn start OR npm install`
* create file called `.env`
* add bot login id. looks like: `LOGIN=ABCDEFGHIJKLMNOPQRSTUVWXYZ.XXXXXX.AAA-bbbbbbbbbbbbbbbbbbbbbbb`
